This Application is used to perform Directory Cleaning.

Description:-

This Application is used to delete duplicate files in given directory
and details are written in Log files and this Log file is send via email
periodically with given time interval.

Command Line Argument Description

DuplicateFileRemoval.py Demo 30 marvellousinfosystem@gmail.com

-DuplicateFileRemoval.py
	Name of Python automation script

-Demo
	Relative path of directory which may contains duplicate files

-30
	Time Interval of script in minutes

-marvellousinfosystem@gmail.com
	Mail Id of the receiver
